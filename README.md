# Financial Market Pricing Analysis

## Project Overview
This project analyzes financial market pricing data to identify daily price movements, pricing exceptions, and potential data quality issues. The goal is to simulate the type of pricing review and exception analysis used in financial operations, investment operations, and FinTech environments.

This project is designed to demonstrate skills in Excel, SQL, Python, financial data analysis, operational risk review, and business reporting.

## Business Problem
Financial institutions rely on accurate daily pricing data for securities, funds, and other financial instruments. Incorrect or unusual prices can create reporting errors, valuation issues, and operational risk.

This project answers questions such as:
- Which securities had the largest daily price movements?
- Which price changes should be flagged for review?
- Are there missing or unusual pricing values?
- What trends can be seen across asset classes?
- How can pricing exceptions be summarized for business users?
